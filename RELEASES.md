# Release Artifact Naming

Use these names for release assets, replacing `{version}` with the release version.

```
Inviska Extract Neo_Windows_ARM_{version}.msi
Inviska Extract Neo_Windows_x64_{version}.msi
Inviska Extract Neo_Linux_ARM_{version}.AppImage
Inviska Extract Neo_Linux_x64_{version}.AppImage
Inviska Extract Neo_macOS_ARM_{version}.zip
Inviska Extract Neo_Source_{version}.zip
Inviska Extract Neo_Source_{version}.tar.gz
```

The macOS zip must contain the `.app` bundle.
